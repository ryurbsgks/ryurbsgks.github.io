Fetch API

1. Fetch API

Fetch API는 JS에서 HTTP를 통한 네트워크 요청을 보내고 응답을 받을 수 있도록 도와주는 API이다.

Fetch API는 Promise의 형식으로 이루어져 있으며 Fetch API가 제공하는 fetch() 메서드로 네트워크 리소스를 쉽게 비동기적으로 가져올 수 있다.

2. fetch()

fetch() 메서드는 첫 번째 매개변수로 URL 두 번째 매개변수로 옵션 객체를 받을 수 있으며 Promise 객체를 반환한다.
반환된 객체는 API 호출이 성공했을 경우 response 객체를 resolve 하고 실패했을 경우에는 error 객체를 reject 한다.

fetch("<url>", <options>)
  .then( (response) => {
    <동작할 코드>
  })
  .catch( (error) => {
    <동작할 코드>
  })

두 번째 매개변수인 옵션 객체에는 HTTP 메서드(method), HTTP 요청 헤더(headers), HTTP 요청 본문(body) 등을 설정할 수 있다.

response 객체로 부터는 HTTP 응답 상태(status), HTTP 응답 헤더(headers), HTTP 응답 본문(body) 등을 읽어올 수 있다.
response 객체가 JSON 형태로 반환되는 경우 json() 메서드를 사용하여 JS 객체로 변환할 수 있다.
