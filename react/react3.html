State

1. State

컴포넌트 내부에서 변할 수 있는 데이터는 변수가 아닌 state를 사용해야 한다.
즉, state는 컴포넌트 내부에서 변할 수 있는 데이터를 의미한다.

2. useState

React에서는 컴포넌트의 state(상태) 관리를 위해 React Hooks에서 제공되는 useState라는 특별한 함수를 사용한다.

useState를 사용하기 위해서는 React로부터 useState를 import 해야 한다.

import { useState } from "react"

3. useState 사용 방법

useState를 사용하는 방법은 useState의 리턴값을 구조 분해 할당으로 선언해 주면 된다.

const [<state>, <setState>] = useState(<initialValue>)

<state>는 현재 state 변수를 의미하며 <setState>는 state 변수를 변경할 수 있는 함수이다.
<initialValue>는 state의 초기값이다.

4. State 변경하기

state 변수를 갱신하려면 <state>를 재할당 하는 것이 아닌 <setState> 함수를 사용해야 한다.

<setState>(<변경할 값>)

React 컴포넌트는 state가 변경되면 새롭게 호출되고 리렌더링 된다.

5. State 위치 정하기

React에서 데이터를 다룰 때는 컴포너트들 간의 상호 관계와 데이터의 역할 및 데이터의 흐름을 고려하여 위치를 설정해야 한다.

하나의 state를 기반으로 여러개의 컴포넌트가 영향을 받는다면 공통 소유 컴포넌트를 찾아 그곳에 상태를 위치해야 한다.
즉, 여러개의 자식 컴포넌트가 하나의 상태에 접근하고자 할 때는 여러 컴포넌트의 공통 부모 컴포넌트에 state를 위치해야 한다.
