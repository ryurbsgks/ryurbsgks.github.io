이벤트 처리

1. 이벤트 처리

React 엘리먼트에서 이벤트를 처리할 때는 일반적인 방식과는 구문상의 차이점이 있다.

React 이벤트 속성은 소문자가 아닌 카멜 케이스(camelCase)를 사용하며 이벤트 핸들러 또한 카멜 케이스를 사용한다.
또한 JSX를 사용하여 문자열이 아닌 함수로 이벤트 핸들러를 전달한다.

const <eventHandler> = () => {
  <동작할 코드>
}

<button onClick={<eventHandler>}><event></button>

2. preventDefault

또 다른 이벤트 처리 차이점으로 React에서는 false를 반환해도 기본 동작을 방지할 수 없다.
기본 동작을 방지하려면 반드시 preventDefault를 사용해야 한다.

<form onSubmit={(e) => e.preventDefault()}>
  <button type="submit">submit</button>
</form>
