수학적 알고리즘

1. 순열

n 개 중에서 일부만을 선택하여 나열하는 것을 순열이라고 하며 순열은 순서를 지키며 나열해야 한다.

일반식 : nPr = n! / (n - r)!

function getPermutation(arr, n, memo){
  if(n === 0){
    console.log(memo);
    return;
  }
  for(let i = 0; i < arr.length; i++){
    let choice = arr[i];
    let newArr = arr.slice();
    newArr.splice(i, 1);
    getPermutation(newArr, n - 1, memo.concat(choice));
  }
}

2. 조합

조합은 n 개 중에서 일부만을 선택하여 나열하지만 순열과 달리 순서를 고려하지 않는다.

일반식 : nCr = n! / (r! * (n - r)!)

function getCombination(arr, n, memo){
  if(n === 0){
    console.log(memo);
    return;
  }
  for(let i = 0; i < arr.length; i++){
    let choice = arr[i];
    let newArr = arr.slice(i + 1);
    getCombination(newArr, n - 1, memo.concat(choice));
  }
}

3. 최소 공배수(LCM)

최소공배수(LCM)란 두 수 이상의 여러 수의 공배수 중 최소인 수를 말한다.

공배수란 두 수 이상의 여러 수의 공통된 배수를 의미한다.

4. 최대 공약수(GCD)

최대공약수(GCD)란 두 수 이상의 여러 수의 공약수 중 최대인 수를 말한다.

공약수란 두 수 이상의 여러 수의 공통된 약수를 의미한다.

5. 유클리드 호제법

유클리드 호제법은 2개의 자연수의 최대 공약수(GCD)를 구하는 방법이다.
2개의 자연수 a, b에 대해서 a를 b로 나눈 나머지를 r이라 하면(단, a > b) a와 b의 최대 공약수(GCD)는 b와 r의 최대 공약수(GCD)와 같다.
이 과정을 반복하여 나머지가 0이 되었을 때 나누는 수가 a와 b의 최대 공약수(GCD)이다.

아래 코드는 유클리드 호제법을 통해 최대 공약수(GCD)와 최소 공배수(LCM)을 구하는 방법이다.

function euclidean(a, b){
  let GCD = (a, b) => a % b === 0 ? b : GCD(b, a % b);
  let LCM = (a, b) => a * b / GCD(a, b);
  console.log(`GCD : ${GCD(a, b)} LCM : ${LCM(a, b)}`)
}

6. 멱집합

어떤 집합이 있을 때 이 집합의 모든 부분집합을 멱집합이라고 한다.

function showPowerset(arr){
  let result = [];
  function powerset(idx, memo){
    if(idx === arr.length){
      result.push(memo);
      return;
    }
    powerset(idx + 1, [...memo, arr[idx]]);
    powerset(idx + 1, memo);
  }
  powerset(0, []);
  return result;
}
