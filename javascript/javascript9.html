배열

1. 배열

JS 배열은 길이도 각 요소의 자료형도 고정되어 있지 않다.
즉 배열의 길이가 언제든지 늘어나거나 줄어들 수 있고 데이터를 연속적이지 않은 곳에 저장할 수 있다.

배열의 구조

대괄호([])를 이용해서 배열을 만들고 각각의 요소는 쉼표(,)로 구분한다.
각 요소들의 위치는 인덱스(index)로 구분하면 인덱스는 0부터 시작한다.

[<element>, <element>, <element>, ...]

배열의 길이

배열의 길이는 인덱스와 달리 1부터 시작하며 length로 길이를 확인할 수 있다.

let newArr = [0, 1, 2, 3];
newArr.length; // expected output : 4

요소에 접근 및 재할당

배열의 요소에 접근하는 방법으로 대괄호([])안에 접근하려는 요소의 인덱스값을 입력한다.

재할당 방법은 배열의 요소에 접근하여 원하는 데이터로 할당해주면 된다.

let newArr = [0, 1, 2, 3];
newArr[2]; // expected output : 2
newArr[2] = "a"; // expected output : [0, 1, "a", 3]

2. 배열 판별 여부

배열의 typeof는 array가 아닌 object로 나온다.
따라서 해당 타입이 정확히 배열인지 판별하기 위해서는 isArray() 메서드를 사용하여 인자가 배열인지 판별한다.

isArray()

isArray() 메서드는 인자가 배열인지 boolean값으로 판별한다.(배열이면 true 배열이 아니면 false)

let newArr = [0, 1, 2, 3];
let newString = "1234";
Array.isArray(newArr); // expected output : true
Array.isArray(newString); // expected output : false

2. 배열 요소 추가 및 삭제

배열의 요소를 추가 및 삭제할 수 있는 pop(), push(), shift(), unshift() 메서드에 대한 설명

pop()

pop() 메서드는 배열에서 마지막 요소를 제거하고 그 요소를 반환한다.

let animals = ["pigs", "goats", "sheep"];
animals.pop(); // expected output : "sheep"
console.log(animals); // expected output : ["pigs", "goats"]

push()

push() 메서드는 배열의 끝에 하나 이상의 요소를 추가하고 배열의 새로운 길이를 반환한다.

문법

arr.push(<element>)

element : 배열의 끝에 추가할 요소

예시

let animals = ["pigs", "goats", "sheep"];
animals.push("cows"); // expected output : 4
console.log(animals); // expected output : ["pigs", "goats", "sheep", "cows"]

shift()

shift() 메서드는 배열에서 첫 번째 요소를 제거하고 제거된 요소를 반환한다.

let animals = ["pigs", "goats", "sheep"];
animals.shift(); // expected output : "pigs"
console.log(animals); // expected output : ["goats", "sheep"]

unshift()

unshift() 메서드는 새로운 요소를 배열의 맨 앞쪽에 추가하고 배열의 새로운 길이를 반환한다.

문법

arr.unshift(<element>)

element : 배열 맨 앞에 추가할 요소

예시

let animals = ["pigs", "goats", "sheep"];
animals.unshift("cows"); // expected output : 4
console.log(animals); // expected output : ["cows", "pigs", "goats", "sheep"]
