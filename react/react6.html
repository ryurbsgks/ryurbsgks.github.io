State 끌어올리기

1. State 끌어올리기

React에서의 데이터 흐름은 하향식이며 단방향 데이터 흐름이 원칙이다.
하지만 하위 컴포넌트에서의 이벤트가 부모의 상태(State)를 바꾸어여만 하는 상황이 올 수 있다.
이를 해결할 수 있는 키워드가 바로 State 끌어올리기(Lifting state up)이다.

상태를 변경시키는 함수를 하위 컴포넌트에 props로 전달하고 이 함수를 하위 컴포넌트가 실행하는 방법으로 state를 끌어올릴 수 있다.

function ParentComponent() {
  const [state, setState] = useState()
  const changeState = () => {
    setState("<newState>")
  }
  return (
    <div>
      <ChildComponent changeState={changeState} />
    </div>
  )
} function ChildComponent({ changeState }) {
  const buttonClick = () => {
    changeState()
  }
  return (
    <div>
      <button onClick={buttonClick}>State 끌어올리기</button>
    </div>
  )
}
