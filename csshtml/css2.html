자주 사용하는 CSS 속성

1. Width, Height

각각 가로 길이, 세로 길이를 의미하는 속성으로 속성값은 숫자 뒤에 절대 단위 또는 상대 단위를 입력하면 된다.

{
  width: 150px;
  height: 100px;
}

2. Margin, Padding

각각 바깥쪽 여백, 안쪽 여백을 의미하는 속성으로 속성값은 숫자 뒤에 절대 단위 또는 상대 단위를 입력하면 된다.

한 개의 값을 입력하면 모든 네 면의 여백을 설정한다.

{ margin(padding): 5px; }

두 개의 값을 입력하면 첫 번째는 위와 아래 두 번째는 왼쪽과 오른쪽 여백을 설정한다.

{ margin(padding): 5px 10px; }

세 개의 값을 입력하면 첫 번째는 위 두 번째는 왼쪽과 오른쪽 세 번째는 아래 여백을 설정한다.

{ margin(padding): 5px 10px 5px; }

네 개의 값을 입력하면 각각 위, 오른쪽, 아래, 왼쪽 순서(위 부터 시계방향)로 여백을 설정한다.

{ margin(padding): 5px 10px 5px 10px; }

margin(padding)-top, margin(padding)-right, margin(padding)-bottom, margin(padding)-left 속성으로 각각 속성을 부여할 수도 있다.

{
  margin(padding)-top: 5px;
  margin(padding)-righ: 10px;
  margin(padding)-bottom: 5px;
  margin(padding)-left: 10px;
}

3. Color

글자의 색상을 변경하는 속성으로 속성값은 주요 색상의 이름, RGB 색상값, 16진수 색상값을 사용할 수 있다.

색상의 이름

표준 색상 명을 입력하여 표현한다.

{ color: black; }

RGB 색상값

RGB의 범위인 0 ~ 255 까지의 숫자로 조합된 색 표현 방식이다.

{ color: rgb(0, 0, 255); }

16진수 색상값

16진수로 표현된 색 표현 방식이다.

{ color: #0000FF; }

4. Font

글자의 폰트를 정의하는 속성이다.

font-style

글꼴의 스타일로 주로 이탤릭체를 설정하기 위해 사용한다.

속성값으로 normal, italic 등이 있다.

{ font-style: italic; }

normal

default 값으로 기본 글꼴 스타일이다.

italic

글꼴 스타일을 이탤릭체로 지정한다.

font-weight

글꼴의 두께로 미리 정의된 단어나 100 ~ 900 사이의 숫자를 통해 설정한다.

미리 정의된 단어로는 lighter(100), normal(400), bold(700), bolder(900)이 있으며 default 값은 normal이다.

{ font-weight: bold; }

font-size

글자 크기를 설정하는 속성으로 글자 크기를 지정하는 방식에는 절대 단위와 상대 단위가 존재한다.

절대 단위

절대 단위는 텍스트의 크기를 명시된 크기 그대로 설정하고자 할 때 사용한다.

절대 단위에는 px, pt 등이 있다.

{ font-size: 15px; }
px(픽셀)

스크린의 픽셀을 기준으로 하는 절대 단위이다.

pt(포인트)

포인트 크기를 의미하는 절대 단위이다.

상대 단위

상대 단위는 텍스트의 크기도 같이 변하는 방식이다.

상대 단위에는 %, em, rem, vw, vh 등이 있다.

{ font-size: 1.5rem; }
%

백분율 단위로 기본 크기를 100%로 설정하고 그에 대한 상대적인 크기를 설정하는 상대 단위이다.

em

해당 글꼴의 기본 크기를 1em으로 설정하고 그에 대한 상대적인 크기를 설정하는 상대 단위이다.

rem

HTML의 폰트 사이즈를 1rem으로 설정하고 그에 대한 상대적인 크기를 설정하는 상대 단위이다.

vw(Viewport Width)

브라우저 너비값의 100분의 1을 나타내는 상대 단위이다. 즉 1vw는 Viewport 넓이의 1%와 같다.

vh(Viewport Height)

브라우저 높이값의 100분의 1을 나타내는 상대 단위이다. 즉 1vh는 Viewport 높이의 1%와 같다.

font-family

글꼴의 종류를 설정하는 속성으로 글꼴의 이름은 따옴표를 붙여서 적용할 수 있다.

여러 글꼴을 사용하는 경우 쉼표(,)로 구분하여 입력할 수 있다.
이때 맨 앞에 있는 글꼴을 우선으로 적용시키며 맨 앞에 있는 글꼴이 사용자의 컴퓨터에 없을 때 그 다음 글꼴을 사용한다.

{ font-family: "SF Pro KR", "MalgunGothic", "Verdana"; }

5. Text-align

텍스트의 정렬 방향을 정의하는 속성으로 속성값은 left, right, center, justify가 있다.
각 속성값마다 left는 왼쪽 정렬, right는 오른쪽 정렬, center는 중앙 정렬, justify는 양쪽 정렬을 의미한다.

{ text-align: center; }

6. background

셀렉터의 배경을 지정하는 속성이다.

{
  background-color: black;
  background-image: url("<image_path>");
  background-repeat: no-repeat;
  background-position: center;
}

background-color

배경색을 의미하며 속성값은 color 속성값과 동일하게 사용한다.

background-image

배경 이미지를 설정하며 주로 이미지 경로를 지정하는 방식으로 사용한다.
경로는 url("<image_path>")와 같은 형식으로 작성한다.

설정된 이미지는 컨테이너에 맞춰 늘어나거나 줄어들지 않고 그대로 표시되며 이미지 보다 컨테이너가 더 크다면 이미지는 반복되어 표시된다.

background-repeat

background-image가 반복되어 출력되는 여부를 지정 할 수 있는 속성으로 속성값으로는 no-repeat, repeat-x, repeat-y, repeat이 있다.
각 속성값마다 no-repeat은 반복하지 않게 설정하며 repeat-x는 가로 방향으로만 반복, repeat-y는 세로 방향으로만 반복, repeat는 반복시키는 값이다.

background-position

일반적으로 background-image는 왼쪽 위부터 이미지를 출력하는데 background-position 속성을 사용하면 이미지의 좌표를 수정 할 수 있게 된다.

띄어쓰기를 기준으로 x좌표, y좌표를 지정하며 픽셀 뿐만 아니라 left, right, top, bottom, center 등의 값도 쓸 수 있다.

7. border

셀렉터의 테두리를 설정하는 속성이다.

{
  border-width: 2px;
  border-color: red;
  border-style: dotted;
}

border-widthn

테두리의 두께를 설정하기 위해 사용한다.

border-color

테두리의 색상으로 속성값은 color 속성값과 동일하게 사용한다.

border-style

테두리의 스타일로 solid, dotted, dashed 등 많은 속성값이 존재한다.

8. border-radius

요소의 테두리를 둥글게 만들어주는 속성이다.

px 단위와 % 단위를 사용할 수 있으며 4개의 모서리를 띄어쓰기를 기준으로 왼쪽 위 부터 시계 방향으로 각자 다른 값을 지정할 수 있다.

{ border-radius: 30px; }

또는

{ border-radius: 10px 10% 10px 10%; }

9. visibility

셀렉터의 가시성을 결정하는 속성으로 속성값은 visible, hidden 등이 있다.

visible은 해당 셀렉터를 보여주는 값이며 hidden은 해당 셀렉터를 숨기는 값이다.
이때 hidden이 가지고 있는 영역은 유지된다.

{ visibility: hidden; }

10. display

요소를 어떻게 보여줄지를 결정하는 속성으로 속성값은 none, block, inline, inline-block이 있다.

none은 요소를 보이지 않게하는 값이며 block은 블록 박스로 inline은 인라인 박스로 inline-block은 block과 inline의 중간 형태로 보여준다.

{ display: none; }
