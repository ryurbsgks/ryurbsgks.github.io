cors 모듈

1. cors 모듈

cors 모듈은 다양한 옵션으로 CORS를 활성화하는 데 사용할 수 있는 Express Middleware를 제공하기 위한 Node.js 모듈이다.

2. cors 설치

cors 모듈을 설치하기 위해 npm install 명령어로 cors를 설치한다.

npm install cors

cors 불러오기

require() 함수를 사용하여 cors 모듈을 불러올 수 있다.

const cors = require('cors')

3. cors 사용 방법

cors의 간단한 사용 방법으로 모든 CORS 요청을 활성화 하는 방법이다.

const express = require('express')
const cors = require('cors')
const app = express()

app.use(cors())

cors 옵션

cors의 기본 옵션 구성은 다음과 같다.

{
  "origin": "*",
  "methods": "GET,HEAD,PUT,PATCH,POST,DELETE",
  "preflightContinue": false,
  "optionsSuccessStatus": 204
}

origin

CORS 헤더의 Access-Control-Allow-Origin을 설정하는 옵션이다.
origin 옵션의 값으로는 boolean, 문자열, RegExp, 배열, 함수, *(와일드카드)가 올 수 있다.

methods

CORS 헤더의 Access-Control-Allow-Methods를 설정하는 옵션으로 쉼표로 구분된 문자열 또는 배열이 올 수 있다.

allowedHeaders

CORS 헤더의 Access-Control-Allow-Headers를 설정하는 옵션으로 쉼표로 구분된 문자열 또는 배열이 올 수 있다.

exposedHeaders

CORS 헤더의 Access-Control-Expose-Headers를 설정하는 옵션으로 쉼표로 구분된 문자열 또는 배열이 올 수 있다.

credentials

CORS 헤더의 Access-Control-Allow-Credentials를 설정하는 옵션이다.
헤더를 전달하려면 true로 설정하고 그렇지 않으면 생략한다.

maxAge

CORS 헤더의 Access-Control-Max-Age를 설정하는 옵션으로 정수만 올 수 있다.

preflightContinue

CORS 실행 전 응답을 다음 핸들러에 전달한다.

optionsSuccessStatus

성공적인 OPTIONS 요청에 사용할 상태 코드를 지정한다.
