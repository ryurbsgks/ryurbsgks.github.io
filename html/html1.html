HTML 기초

1. HTML

HTML은 HyperText Markup Language의 약자로 웹 페이지를 구성하는 마크업 언어이다.

HTML에는 수많은 Tag와 Input type들이 존재한다.
기본 구조는 크게 head와 body영역으로 구성되며 각 영역에 세부 요소들의 tag와 Semantic tag들이 들어온다.

Tag

Tag는 Start tag(Opening tag)와 End tag(Closing tag) 그리고 그 사이의 Content로 구성된다.

Semantic tag

Semantic의 사전적 의미는 "의미론적인" 이라는 뜻이다. 즉 의미론적인 태그를 뜻한다.

HTML작성시 div나 span등 그 이름만 들어서는 그 내용이 무엇을 뜻하는지 한번에 알아보기가 힘들다.
HTML5로 넘어오면서 이 문제를 해결하기 위해 각 태그들이 무엇을 뜻하는지 명시적으로 표현하기 위해 Semantic tag를 활용하기 시작했다.
Semantic tag의 사용 장점은 각 태그들이 무엇을 의미하는지 한눈에 알아보기 쉽도록 구현 할 수 있다는 것이다.

Element

element는 HTML을 문서형식으로 구성할 때 특정 의미를 표현하는 약속된 단어이다.

tag와 차이점은 img자체는 element(요소)이며 요소를 <>로 깜산 <img>는 tag(태그)이다.
