Signup 컴포넌트

1. Signup 컴포넌트

Signup 컴포넌트는 회원가입을 위한 정보를 입력하는 컴포넌이다.

pages 폴더에 Signup.js 파일이 있으며 component 폴더에 Signup.js 파일이 있다.

Font Awesome 패키지 설치

Font Awesome을 사용하기 위해 npm을 사용하여 아래와 같은 패키지를 설치하였다.

npm i --save @fortawesome/fontawesome-svg-core
npm install --save @fortawesome/free-solid-svg-icons
npm install --save @fortawesome/react-fontawesome

src/App.js

추가 된 코드는 아래와 같다.

import Signup from "./pages/Signup";

<Route path="/signup" element={<Signup />} />

전체 코드는 아래와 같다.

import './App.css';
import { BrowserRouter, Routes, Route } from "react-router-dom";
import Main from "./pages/Main";
import Signup from "./pages/Signup";

function App() {
  return (
    <BrowserRouter>
      <Routes>
          <Route path="/" element={<Main />} />
          <Route path="/signup" element={<Signup />} />
      </Routes>
    </BrowserRouter>
  );
}

export default App;
