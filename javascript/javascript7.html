문자열

1. 문자열

문자열은 String 타입으로 문자열 선언 방법은 따옴표 안에 원하는 문자열을 입력하거나 템플릿 리터럴을 사용하는 방법이 있다.

따옴표 사용

가장 기본적인 문자열 선언 방식으로 큰따옴표("")와 작은따옴표('')를 이용하여 문자열을 선언할 수 있다.

템플릿 리터럴(Template literals)

템플릿 리터럴은 내장된 표현식을 허용하는 문자열 리터럴로 여러 줄로 이루어진 문자열과 문자 보간 기능을 사용할 수 있다.

사용 방법은 따옴표 대신 백틱(``)을 사용하여 백틱안에 문자열을 입력하여 사용 한다.
또한 플레이스 홀더(${})를 사용하여 표현식도 나타낼 수 있다.

템플릿 리터럴에서 줄 바꿈은 \n 또는 엔터를 사용하여 표현할 수 있다.

2. length

length 프로퍼티(property)는 해당 문자열의 길이를 반환한다.

<string>.length

3. index

문자열에서 특정 위치를 가리키는 숫자를 index라고 한다.

문자열에서 index는 읽기 전용(read-only)으로 접근은 가능하지만 값을 재할당 할 수는 없다.

<string>[<index>]

4. 문자열 합치기

문자열을 합치는 방법에는 + 연산자를 사용하거나 concat() 메서드를 사용하는 방법이 있다.

+ 연산자

합치려고 하는 문자열들을 + 연산자를 사용하여 합칠 수 있다.

concat()

concat() 메서드는 매개변수로 전달된 모든 문자열을 호출 문자열에 붙인 새로운 문자열을 반환한다.

<string>.concat(<string>)

5. 문자열 타입 변경

문자열의 타입을 변경하는 방법으로 상황에 따라 문자열을 숫자로 사용하거나 숫자를 문자열로 사용해야하는 경우가 있다.
주의할 점은 원본 변수의 타입이 완전히 변경되는 것이 아닌 결과값만 타입이 변경된다는 것이다.

문자열을 숫자로 변경

문자열을 숫자로 변경하는 방법에는 Number() 생성자, Math 객체의 메서드, parseInt() 또는 parseFloat() 함수, 문자열에 숫자를 곱하는 방법이 있다.

Number()

Number() 생성자는 해당 변수를 Number 타입으로 변환하여 반환한다.

Number(<variable>)

Math 객체의 메서드

Math 객체의 메서드 중 abs(), floor(), ceil() 메서드를 사용하여 문자열을 숫자로 반환할 수 있다.

abs()

abs() 메서드는 주어진 값의 절대값을 반환하는 메서드이다.

Math.abs(<variable>)
floor()

floor() 메서드는 주어진 값보다 작거나 같은 수 중에서 가장 큰 정수를 반환하는 메서드이다.
즉 내림과 같다.

Math.floor(<variable>)
ceil()

ceil() 메서드는 주어진 값보다 크거나 같은 수 중에서 가장 작은 정수를 반환하는 메서드이다.
즉 올림과 같다.

Math.ceil(<variable>)

parseInt()

parseInt() 함수는 주어진 값을 파싱하여 특정 진수의 정수를 반환한다.

parseInt(<string>, <radix>)

radix는 string을 숫자로 바꿀때 사용하는 진수로 2 ~ 36 까지의 정수로 생략 가능하다.

parseFloat()

parseFloat() 함수는 주어진 값을 문자열로 변환한 후 부동소수점 실수로 파싱해 반환한다.

parseFloat(<string>)

숫자를 곱하는 방법

문자열에 숫자를 곱하면 결과값이 Number 타입으로 변환하여 반환한다.

숫자를 문자열로 변경

숫자를 문자열로 변경하는 방법에는 String() 생성자, toString() 메서드, 빈 문자열을 더하는 방법이 있다.

String()

String() 생성자는 해당 변수를 String 타입으로 변환하여 반환한다.

String(<variable>)

toString()

toString() 메서드는 주어진 값을 문자열로 변환하여 반환한다.

<variable>.toString()

빈 문자열을 더하는 방법

숫자에 빈 문자열을 더하면 결과값이 String 타입으로 변환하여 반환한다.
