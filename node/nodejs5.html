path 모듈

1. __filename & __dirname

path 모듈을 설명하기 이전에 path 모듈에서도 많이 사용하는 __filename과 __dirname에 대해서 간단히 알아보자

Node.js는 __filename과 __dirname으로 경로에 대한 정보를 제공한다.

__filename

파일명을 포함한 절대 경로를 나타낸다.

__dirname

파일명을 제외한 절대 경로를 나타낸다.
즉, 현재 파일의 상위 디렉토리의 절대 경로를 나타낸다.

2. path 모듈

path 모듈은 파일 및 디렉토리의 경로 작업을 위한 기능을 제공하는 모듈이다.

path 모듈 불러오기

아래와 같이 require() 함수를 사용하여 path 모듈을 불러올 수 있다.

const path = require('path')

3. path 모듈 메서드

path 모듈에는 많은 메서드가 내장되어 있지만 여기에서는 join(), normalize(), resolve() 메서드만 다룬다.
더 많은 모듈에 대해서 알고 싶다면 path 모듈 공식 API 문서를 참고하자

path.join()

path.join() 메서드는 플랫폼별 구분 기호를 구분 기호로 사용하여 주어진 모든 경로를 합친 후 문자열 형태로 반환한다.

path.join(<path>, <path>, ...)

path.normalize()

path.normalize() 메서드는 주어진 경로에서 ".." 및 "." 등을 수정하여 문자열 형태로 반환한다.

path.normalize(<path>)

path.resolve()

path.resolve() 메서드는 플랫폼별 구분 기호를 구분 기호로 사용하여 주어진 모든 경로를 합친 후 문자열 형태로 반환한다.
주의할점은 주어진 매개변수를 읽는 순서가 오른쪽에서 왼쪽 순이며 합쳐서 반환할 때는 왼쪽에서 오른쪽으로 합쳐서 반환한다.
매개변수를 읽는 중 "/"를 만나면 절대경로로 인식하여 나머지 매개변수들을 무시한다.

path.resolve(<path>, <path>, ...)
