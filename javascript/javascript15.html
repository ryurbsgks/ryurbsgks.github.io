Class & Instance

1. 클래스(Class)

클래스는 객체 지향 프로그래밍에서 특정 객체를 생성하기 위해 속성과 메서드를 정의하는 일종의 틀로 객체를 정의하기 위한 속성과 메서드로 구성된다.

2. 클래스 선언

클래스를 선언하는 방식은 일반적인 함수를 정의하듯 만드는 방법과 class 키워드를 이용하여 정의하는 방법이 있다.

클래스를 선언할때 함수와 구별을 위해 대문자 그리고 일반명사로 만든다.

function <ClassName>(<parameter>) {
  this.<parameter> = <parameter>
}

<ClassName>.prototype.<method> = function () {
  this.<parameter>
  <동작할 코드>
}

ES6 부터는 class라는 키워드를 이용해서 정의할 수 있다.

class <ClassName> {
  constructor(<parameter>) {
    this.<parameter> = <parameter>
  }

  <method>() {
    this.<parameter>
    <동작할 코드>
  }
}

생성자(constructor)

constructor() 메서드는 class로 생성된 객체를 생성하고 초기화하기 위한 메서드로 constructor() 메서드는 class 안에 한 개만 존재할 수 있다.

여기서 this는 인스턴스 객체를 의미한다.
매개변수(parameter)로 넘어온 값들은 인스턴스 생성시 지정하는 값이며 this에 할당한다는 것은 만들어진 인스턴스에 해당 값들을 부여하겠다는 의미이다.

메서드(method)

해당 클래스에서 작동하는 기능들을 사용하기 위해 정의된 함수를 메서드라고 부른다.

ES6 이전에는 prototype이라는 키워드를 사용해야 메서드를 정의할 수 있었지만 ES6에서는 생성자 함수와 함께 class 키워드 안쪽에 묶어서 정의한다.

메서드에서 해당 클래스의 매개변수를 사용하려면 매개변수 앞에 this를 붙여서 사용하면 된다.

3. 클래스 상속

부모 클래스에서 자식 클래스로 부모 클래스의 속성과 메서드를 물려주는 과정을 상속이라고 하며 JS에서는 extends와 super 키워드를 사용하여 상속을 구현할 수 있다.

class <ChildClassName> extends <ParentClassName> {
  constructor(<parentParameter>, <childParameter>) {
    super(<parentParameter>)
    this.<childParameter> = <childParameter>
  }
}

extends

extends 키워드는 클래스를 다른 클래스의 자식으로 만들기 위해 class 선언시에 사용된다.

super

super 키워드는 부모 클래스의 함수를 호출할 때 사용된다.

4. 인스턴스(Instance)

클래스의 속성과 메서드를 담고 있는 객체를 인스턴스라고 부른다.

5. 인스턴스 생성

인스턴스를 생성할 때는 new 키워드를 사용하여 생성한다.

let <instanceName> = new <ClassName>(<parameter>)

이렇게 생성된 <instanceName> 인스턴스는 <ClassName>의 고유한 속성과 메서드를 갖게 되며 속성과 메서드 사용 방법은 아래와 같다.

<instanceName>.<parameter> <instance>.<method>()
