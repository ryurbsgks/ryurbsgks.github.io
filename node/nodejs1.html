Node.js

1. Node.js

Node.js라는 새로운 JavaScript 런타임의 등장으로 JavaScript가 브라우저가 아닌 곳에서 실행될 수 있게 되었다.
이로인해 JavaScript를 이용해서 웹 페이지뿐만 아니라 서버와 같은 다른 프로그램을 만들 수 있게 되었다.

CLI에서 node 명령어를 입력하면 작성한 코드가 Node.js라는 런타임에서 실행된다.

node <file_name>

런타임

런타임이란 프로그래밍 언어가 구동되는 환경 즉, 프로그램이 동작할 때 프로그램이 동작하는 곳을 말한다.

2. nvm

nvm은 Node Version Manager의 약자로 Node.js의 다양한 버전을 쉽게 설치하고 사용할 수 있다.

nvm 설치

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

버전이 최신 버전으로 바뀌어 있을 수 있으니 nvm 공식문서를 한번 확인해 보는 것도 좋다.

만일 Command 'wget' not found 메세지와 함께 설치가 진행되지 않는 경우 Package Manager를 이용해 wget을 설치한 후 진행한다.

sudo apt update
sudo apt install wget

nvm 설치 확인

터미널을 다시 시작한 이후 --version 명령어를 이용하여 nvm 버전을 확인한다.

nvm --version

버전이 잘 나온다면 nvm 설치가 성공적으로 되었다.

Linux에서 설치 이후 nvm: command not found 가 표시되면 현재 터미널을 닫고 새 터미널을 연 다음 다시 확인하거나 해당 셸(shell)에 대한 아래 명령어를 입력한다.

bash : source ~/.bashrc

zsh : source ~/.zshrc

ksh : . ~/.profile

nvm 간단 사용법

ls 명령어는 현재 nvm을 통해 설치한 Node version들이 출력된다.

nvm ls

use 명령어는 사용 중인 Node version을 다른 버전으로 변경하고 싶을 때 use 명령어 다음에 원하는 Node version을 입력해 주면 된다.

nvm use <version_number>

3. Node.js 설치

Node.js는 여러가지 방법으로 설치할 수 있지만 Node.js의 다양한 버전을 쉽게 설치할 수 있는 nvm을 위에서 배웠으니 nvm을 통한 설치를 한다.

nvm을 통한 설치 방법은 install 명령어로 매우 간단하다.
install 명령어 다음에 설치하려는 Node.js 버전을 적어주면 된다.

nvm install <version>

Node.js 설치 확인

성공적으로 Node.js가 설치되었으면 아래 명령어로 Node.js 버전을 확인한다.

node -v

터미널에 버전 정보가 출력되면 Node.js 설치가 성공적으로 되었다.
