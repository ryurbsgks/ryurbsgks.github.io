React Redux

1. React Redux

React Redux는 Redux의 공식 React UI 바인딩 레이어이다.
이를 통해 React 구성 요소가 Redux Store에서 데이터를 읽고 Store에 Action을 전달하여 상태를 업데이트할 수 있다.

2. React Redux 설치

기존 React 앱에서 React Redux를 사용하려면 npm install을 사용하여 dependency 설치를 한다.
--save 옵션을 넣지 않아도 dependency 설치가 된다.

npm install react-redux

Redux Store 설정

React Redux 설치가 완료되면 앱에 Redux Store를 설정 해야 한다.

import React from 'react'
import { render } from 'react-dom'
import { Provider } from 'react-redux'
import { createStore } from 'redux'
import rootReducer from './reducers'
import App from './components/App'

const store = createStore(rootReducer)

render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
)

createStore()

createStore() 메서드는 앱의 전체 상태 트리를 보유 하는 Redux Store를 만드는 메서드이다.

createStore(<reducer>, <preloadedState>, <enhancer>)

<reducer>는 현재 상태 트리와 처리할 작업이 주어지면 다음 상태 트리를 반환하는 축소 함수이다.

<preloadedState>는 초기 상태로 선택적으로 앱의 서버에서 상태를 수화하거나 이전에 직렬화된 사용자 세션을 복원하도록 지정할 수 있다.

<enhancer>는 미들웨어, 시간 여행, 지속성 등과 같은 third-party 기능으로 저장소를 향상시키기 위해 선택적으로 지정할 수 있다.

Provider

React Redux에는 <Provider> 컴포넌트가 포함되어 있어 앱의 나머지 부분에서 Redux Store를 사용할 수 있다.

3. Redux Hooks

React Redux는 React 컴포넌트가 Redux Store와 상호 작용할 수 있도록 하는 한 쌍의 사용자 지정 React Hook을 제공한다.

Redux Hooks에서는 크게 useSelector(), useDispatch() 2가지의 메서드를 기억하면 된다.

useSelector()

useSelector() 메서드는 컴포넌트와 Store의 상태(State)를 연결하는 메서드로 컴포넌트에서 useSelector() 메서드를 통해 Store의 상태에 접근할 수 있다.

import { useSelector } from 'react-redux'

const <variable> = useSelector( (state) => state.<stateValue>)

useSelector() 메서드의 전달인자로는 콜백 함수를 받으며 콜백 함수의 전달인자로는 state 값이 들어간다.

useDispatch()

useDispatch() 메서드는 Dispatch 함수를 Hooks로 사용하는 메서드로 Action 객체를 Reducer로 전달해 주는 메서드이다.

import { useDispatch } from 'react-redux'

const dispatch = useDispatch()

dispatch(<ActionObj>)

useDispatch() 메서드의 전달인자로는 Action 객체를 받는다.
