자주 사용 하는 메서드

1. Math 객체의 메서드

자주 사용하는 Math 객체의 메서드 정리

pow()

주어진 밑 값을 주어진 지수 값으로 제곱한 값을 반환한다.

Math.pow(<base>, <exponent>)

base는 밑 값을 의미하며 exponent는 밑 을 제곱하기 위해 사용하는 지수를 의미한다.

floor()

주어진 값보다 작거나 같은 수 중에서 가장 큰 정수를 반환한다.
즉, 내림과 같다.

Math.floor(<variable>)

ceil()

주어진 값보다 크거나 같은 수 중에서 가장 작은 정수를 반환한다.
즉, 올림과 같다.

Math.ceil(<variable>)

round()

주어진 값을 반올림한 수와 가장 가까운 정수 값을 반환한다.

Math.round(<variable>)

abs()

주어진 값의 절대값을 반환한다.

Math.abs(<variable>)

sqrt()

주어진 값의 제곱근을 반환한다.

Math.sqrt(<variable>)

max()

입력값으로 받은 숫자 중 가장 큰 숫자를 반환한다.

Math.max(<...variable>)

min()

입력값으로 받은 숫자 중 가장 작은 숫자를 반환한다.

Math.min(<...variable>)

random()

0 이상 1 미만 사이의 난수를 반환한다.

Math.random()

2. String 객체의 메서드

자주 사용하는 String 객체의 메서드 정리

includes()

특정 문자열이 해당 문자열에 포함되어 있는지를 판별하고 결과를 true 또는 false 로 반환한다.

<str>.includes("<searchString>", <index>)

searchString은 문자열에서 찾을 특정 문자열이며 index는 searchString을 찾기 시작할 위치로 생략 가능하다.

indexOf()

해당 문자열에서 주어진 값과 일치하는 첫 번째 인덱스를 반환한다.
일치하는 값이 없으면 -1을 반환한다.

<str>.indexOf("<searchString>", <index>)

searchString은 문자열에서 찾을 특정 문자열이며 index는 searchString을 찾기 시작할 위치로 생략 가능하다.

toUpperCase()

해당 문자열을 대문자로 변환해서 반환한다.
원래의 문자열에 영향을 주지는 않는다.

<str>.toUpperCase()

toLowerCase()

해당 문자열을 소문자로 변환해서 반환한다.
원래의 문자열에 영향을 주지는 않는다.

<str>.toLowerCase()

trim()

해당 문자열의 양 끝 공백을 제거한다.

<str>.trim()

concat()

문자열 인수를 해당 문자열에 붙인 새로운 문자열을 반환한다.

<str>.concat("<string>")

split()

해당 문자열을 지정한 구분자를 이용하여 여러 개의 문자열로 나누어 배열로 반환한다.

<str>.split("<separator>")

separator는 원본 문자열을 나누어야 할 부분을 나타내는 문자열로 생략 가능하며 생략시 원본 문자열을 유일한 원소로 가지는 배열을 반환한다.

slice()

해당 문자열의 일부를 추출하여 반환한다.

<str>.slice(<beginIndex>, <endIndex>)

beginIndex는 추출을 시작할 인덱스로 음수이면 <str>.length + <beginIndex>로 취급된다.
또한 beginIndex가 <str>.length 보다 크거나 같은 경우 빈 문자열을 반환한다.

endIndex는 추출을 종료할 인덱스로 그 직전까지 추출한다.
endIndex는 생략 가능하며 생략시 문자열 마지막까지 추출한다.
또한 endIndex가 음수이면 beginIndex와 마찬가지로 <str>.length + <endIndex>로 취급된다.

substring()

해당 문자열의 일부를 추출하여 반환한다.

<str>.substring(<beginIndex>, <endIndex>)

beginIndex는 추출을 시작할 인덱스이며 endIndex는 추출을 종료할 인덱스로 그 직전까지 추출한다.
endIndex는 생략 가능하며 생략시 문자열 마지막까지 추출한다.

만약 beginIndex가 endIndex와 같을 경우 substring()은 빈 문자열을 반환하고 beginIndex가 endIndex 보다 큰 경우 substring() 메서드는 두 개의 인수를 바꾼 듯 작동하게 된다.

3. Array 객체의 메서드

자주 사용하는 Array 객체의 메서드 정리

includes()

해당 배열이 특정 요소를 포함하고 있는지 판별하고 결과를 true 또는 false 로 반환한다.

<arr>.includes(<searchElement>, <index>)

searchElement는 배열에서 찾을 특정 요소이며 index는 searchElement를 찾기 시작할 위치로 생략 가능하다.

indexOf()

해당 배열에서 주어진 요소와 일치하는 첫 번째 인덱스를 반환한다.
일치하는 요소가 없으면 -1을 반환한다.

<arr>.indexOf(<searchElement>, <index>)

searchElement는 배열에서 찾을 특정 요소이며 index는 searchElement를 찾기 시작할 위치로 생략 가능하다.

find()

해당 배열에서 주어진 판별 함수를 만족하는 첫 번째 요소를 반환한다.
일치하는 요소가 없으면 undefined를 반환한다.

<arr>.find(<callback>)

callback

배열의 각 요소에 실행할 함수로 element, index, array 3개의 매개변수를 받는다.

element는 콜백 함수에서 처리할 현재 요소이다.

index는 콜백 함수에서 처리할 현재 요소의 인덱스이며 array는 find() 메서드를 호출한 배열로 index와 array는 생략 가능하다.

some()

해당 배열의 어떤 요소라도 주어진 판별 함수를 통과하는지 확인하여 통과하면 true 통과하지 못하면 false를 반환한다.

<arr>.some(<callback>)

callback

배열의 각 요소에 실행할 함수로 element, index, array 3개의 매개변수를 받는다.

element는 콜백 함수에서 처리할 현재 요소이다.

index는 콜백 함수에서 처리할 현재 요소의 인덱스이며 array는 some() 메서드를 호출한 배열로 index와 array는 생략 가능하다.

every()

해당 배열의 모든 요소가 주어진 판별 함수를 통과하는지 확인하여 통과하면 true 통과하지 못하면 false를 반환한다.

<arr>.every(<callback>)

callback

배열의 각 요소에 실행할 함수로 element, index, array 3개의 매개변수를 받는다.

element는 콜백 함수에서 처리할 현재 요소이다.

index는 콜백 함수에서 처리할 현재 요소의 인덱스이며 array는 every() 메서드를 호출한 배열로 index와 array는 생략 가능하다.

filter()

해당 배열에서 주어진 판별 함수를 만족하는 모든 요소를 모아 새로운 배열로 반환한다.

<arr>.filter(<callback>)

callback

배열의 각 요소에 실행할 함수로 element, index, array 3개의 매개변수를 받는다.

element는 콜백 함수에서 처리할 현재 요소이다.

index는 콜백 함수에서 처리할 현재 요소의 인덱스이며 array는 filter() 메서드를 호출한 배열로 index와 array는 생략 가능하다.

sort()

해당 배열의 요소들을 정렬한 후 그 배열을 반환한다.

<arr>.sort(<callback>)

callback

정렬 순서를 정의하는 함수로 생략 가능하며 생략하면 해당 요소를 문자열로 변환한 유니코드값에 따라 정렬된다.

callback 함수가 제공되면 배열 요소는 callback 함수의 반환값에 따라 정렬된다.
callback 함수의 매개변수가 순서대로 a와 b이며 a와 b가 비교되는 두 요소라면 callback 함수의 반환값이 0보다 작은 경우 a가 먼저 오고 0보다 큰 경우 b가 먼저 온다.
만약 callback 함수의 반환값이 0이면 a와 b를 변경하지 않는다.

fill()

해당 배열의 요소들을 정적인 값 하나로 채운 후 해당 배열을 반환한다.

<arr>.fill(<value>, <beginIndex>, <endIndex>)

value는 해당 배열의 요소들을 채울 값이다.

beginIndex는 값을 채우기 시작할 인덱스이며 endIndex는 종료할 인덱스로 그 직전까지 값을 채운다.
beginIndex와 endIndex는 생략 가능하며 생략하면 beginIndex의 default 값은 0, endIndex의 default 값은 <arr>.length 이다.

concat()

인수로 주어진 배열이나 값을 해당 배열에 합쳐서 새 배열을 반환한다.

<arr>.concat(<argument>)

join()

해당 배열의 모든 요소를 연결해 하나의 문자열로 만들어 반환한다.

<arr>.join("<separator>")

separator는 배열의 각 요소를 구분할 문자열로 생략 가능하며 생략시 배열의 요소들을 쉼표로 구분한다.

slice()

해당 배열의 특정 부분을 추출하여 추출한 요소를 포함한 새로운 배열을 반환한다.

<arr>.slice(<beginIndex>, <endIndex>)

beginIndex는 추출을 시작할 인덱스로 음수인 경우 배열의 끝에서부터의 길이를 나타낸다.
또한 beginIndex가 배열의 길이보다 큰 경우에는 빈 배열을 반환한다.

endIndex는 추출을 종료할 인덱스로 그 직전까지 추출한다.
endIndex가 음수인 경우 <arr>.length + <endIndex>로 취급된다.
또한 endIndex가 배열의 길이보다 큰 경우에는 <arr>.length로 취급된다.

beginIndex와 endIndex는 생략 가능하며 생략시 beginIndex의 default 값은 0, endIndex의 default 값은 <arr>.length가 된다.

splice()

해당 배열의 기존 요소를 삭제 또는 수정하거나 새 요소를 추가하여 배열의 내용을 변경하는 메서드로 삭제한 요소를 담은 배열을 반환한다.
만약 요소를 삭제하지 않았으면 빈 배열을 반환한다.

<arr>.splice(<beginIndex>, <deleteCount>, <variable>)

beginIndex는 해당 배열의 변경을 시작할 인덱스로 음수인 경우 배열의 끝에서부터의 길이를 나타낸다.
또한 beginIndex가 배열의 길이보다 큰 경우에는 <arr>.length로 취급된다.

deleteCount는 해당 배열에서 제거할 요소의 개수이다.

variable은 해당 배열에 추가할 요소이다.

deleteCount와 variable은 생략 가능하며 생략시 deleteCount는 beginIndex부터의 모든 요소를 제거하며 variable은 추가할 요소가 없으므로 splice() 메서드는 요소를 제거하기만 한다.

forEach()

forEach() 메서드는 해당 배열의 각 요소에 주어진 함수를 실행하는 메서드로 undefined를 반환한다.

<arr>.forEach(<callback>)

callback

배열의 각 요소에 실행할 함수로 element, index, array 3개의 매개변수를 받는다.

element는 콜백 함수에서 처리할 현재 요소이다.

index는 콜백 함수에서 처리할 현재 요소의 인덱스이며 array는 forEach() 메서드를 호출한 배열로 index와 array는 생략 가능하다.

map()

해당 배열의 각 요소에 주어진 함수를 실행하고 그 결과를 요소로 갖는 배열을 반환한다.

<arr>.map(<callback>)

callback

배열의 각 요소에 실행할 함수로 element, index, array 3개의 매개변수를 받는다.

element는 콜백 함수에서 처리할 현재 요소이다.

index는 콜백 함수에서 처리할 현재 요소의 인덱스이며 array는 map() 메서드를 호출한 배열로 index와 array는 생략 가능하다.

reduce()

해당 배열의 각 요소에 주어진 함수를 실행하고 결과값을 누적하여 최종적으로 누적 계산의 결과값 하나를 반환한다.

<arr>.reduce(<callback>, <initialValue>)

callback

배열의 각 요소에 실행할 함수로 accumulator, currentValue, currentIndex, array 4개의 매개변수를 받는다.

accumulator는 callback 함수의 반환값을 누적하는 값이다.

currentValue는 콜백 함수에서 처리할 현재 요소이다.

currentIndex는 콜백 함수에서 처리할 현재 요소의 인덱스이며 생략 가능하다.
initialValue를 제공한 경우 0 부터 시작하며 initialValue를 제공하지 않은 경우 1 부터 시작한다.

array는 reduce() 메서드를 호출한 배열로 생략 가능하다.

initialValue

initialValue는 callback 함수가 처음 호출될 때 accumulator가 초기화 되는 값으로 생략 가능하며 생략시 배열의 첫 번째 요소를 사용한다.
