셀렉터의 종류

1. 전체 셀렉터

HTML 문서 내의 모든 요소를 선택한다.

* {
  <CSS_content>
}

2. Tag 셀렉터

해당 태그만 선택한다.

<tag_name> {
  <CSS_content>
}

3. Id 셀렉터

해당 id를 갖는 요소를 선택한다.(id는 문서 내에 단 하나의 요소에만 적용할 수 있다는 사실을 기억하자)

#<id_name> {
  <CSS_content>
}

4. Class 셀렉터

해당 class를 갖는 요소들을 선택한다.

.<class_name> {
  <CSS_content>
}

5. 후손 셀렉터

선택한 상위 셀렉터는 상위 요소가 되고 상위 요소의 모든 하위 요소 중 선택한 하위 셀렉터와 일치하는 하위 요소를 선택한다.(문법에서 앞에 오는 셀렉터가 상위 셀렉터 뒤에 오는 셀렉터가 하위 셀렉터)

<selector> <selector> {
  <CSS_content>
}

6. 자식 셀렉터

후손 셀렉터와 달리 모든 하위 요소가 아닌 자식 요소 중 선택한 하위 셀렉터와 일치하는 하위 요소를 선택한다.

<selector> > <selector> {
  <CSS_content>
}

7. 일반 형제 셀렉터

2개의 셀렉터(앞 셀렉터, 뒤 셀렉터)가 같은 부모를 가지고 앞 셀렉터와 일치하는 요소 뒤에 있는 모든 요소 중 뒤 셀렉터와 일치하는 요소들을 선택한다.

<selector> ~ <selector> {
  <CSS_content>
}

8. 인접 형제 셀렉터

일반 형제 셀렉터와 달리 앞 셀렉터와 일치하는 요소 뒤에 있는 모든 요소 중 뒤 셀렉터와 일치하는 첫 번째로 오는 요소만 선택한다.

<selector> + <selector> {
  <CSS_content>
}

9. 가상 클래스 셀렉터

셀렉터 뒤에 특정 이벤트를 붙여 해당 셀렉터의 요소가 특정 이벤트일 경우 선택한다.

hover

마우스가 해당 요소 위에 있을 때

<selector>:hover {
  <CSS_content>
}

active

마우스가 해당 요소를 클릭하는 순간부터 떼는 순간 까지

<selector>:active {
  <CSS_content>
}

focus

마우스가 해당 요소를 클릭했을 때

<selector>:focus {
  <CSS_content>
}

link

방문하지 않은 링크일 때

<selector>:link {
  <CSS_content>
}

visited

방문한 링크일 때

<selector>:visited {
  <CSS_content>
}

checked

해당 셀렉터가 체크된 상태일 때

<selector>:checked {
  <CSS_content>
}

enabled

해당 셀렉터가 사용 가능한 상태일 때

<selector>:enabled {
  <CSS_content>
}

disabled

해당 셀렉터가 사용 불가능한 상태일 때

<selector>:disabled {
  <CSS_content>
}

10. 부정 셀렉터

앞에 오는 셀렉터와 일치하는 요소 중에서 not 뒤에 오는 셀렉터와 일치하는 요소가 아닌 요소를 선택

<selector>:not(<selector>) {
  <CSS_content>
}

11. 속성(Attribute) 셀렉터

해당 셀렉터의 특정 속성이나 특정 속성값을 가지고 있는 요소를 선택

<selector>[<attribute>] {
  <CSS_content>
}

or

<selector>[<attribute>="<value>"] {
  <CSS_content>
}
