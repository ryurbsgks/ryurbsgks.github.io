nodemon

1. nodemon

nodemon은 디렉토리의 파일 변경이 감지되면 노드 응용 프로그램을 자동으로 다시 시작하여 Node.js 기반 응용 프로그램을 개발하는 데 도움이 되는 도구이다.

2. nodemon 설치

nodemon은 로컬 설치를 사용하면 CLI 환경에서 nodemon 명령어를 직접 사용할 수 없으므로 -g 옵션을 붙여 global 설치를 통해 시스템 경로에 전역으로 설치한다.

npm install -g nodemon

또는 --save-dev 옵션을 통해 로컬 설치를 하여 npm script에서 nodemone 명령어를 사용한다.

npm install --save-dev nodemon

3. nodemon 사용 방법

기존에 Node.js를 실행할 때 node 명령어를 사용하였다면 nodemon은 nodemon 명령어를 사용하면 된다.

nodemon <실행할 파일명>

또는 package.json scripts 항목에 nodemon을 설정하여 npm script로 실행한다.

{
  "scripts": {
    "start": "nodemon <실행할 파일명>"
  }
} npm run start

or

npm start

nodemon 종료

nodemon을 종료하고 싶으면 ctrl + c 키를 입력하여 nodemon을 종료한다.

nodemon 수동 재시작

nodemon이 실행되는 동안 수동으로 서버를 다시 시작하고 싶은 경우 nodemon을 종료했다가 다시 시작하는 대신 rs 명령어를 입력하고 enter키를 누르면 nodemon이 서버를 재시작한다.
