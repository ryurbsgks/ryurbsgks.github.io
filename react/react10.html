Font Awesome

1. Font Awesome

Font Awesome은 자주 사용되는 아이콘을 쉽게 접근할 수 있도록 모아놓은 오픈소스 라이브러리이다.

패키지 설치

React에서 Font Awesome을 사용하기 위해 npm을 사용하여 아래와 같은 패키지를 설치한다.

npm i --save @fortawesome/fontawesome-svg-core
npm install --save @fortawesome/free-solid-svg-icons
npm install --save @fortawesome/react-fontawesome

Font Awesome 사용 방법

import { FontAwesomeIcon } from '@fortawesome/react-fontawesome'
import { <사용할 아이콘> } from '@fortawesome/free-solid-svg-icons'

<FontAwesomeIcon icon={<사용할 아이콘>} />

<사용할 아이콘>의 변수명은 fa + <아이콘 이름>으로 설정하며 이때 아이콘의 이름은 카멜 표기법을 사용해야 한다.
