자주 사용 하는 메서드

1. Math 객체 메서드

자주 사용하는 Math 객체의 메서드 정리

Math.pow()

Math.pow() 함수는 base 에 exponent를 제곱한 값을 반환한다.

문법

Math.pow(<base>, <exponent>)

base : 밑 값
exponent : 밑 을 제곱하기 위해 사용하는 지수

예시

Math.pow(7, 3); // expected output : 343

Math.floor()

Math.floor() 함수는 주어진 숫자와 같거나 작은 정수 중에서 가장 큰 수를 반환한다.(내림)

문법

Math.floor(<x>)

x : 숫자

예시

Math.floor(5.95); // expected output : 5

Math.ceil()

Math.ceil() 함수는 주어진 숫자보다 크거나 같은 숫자 중 가장 작은 숫자를 반환한다.(올림)

문법

Math.ceil(<x>)

x : 숫자

예시

Math.ceil(5.95); // expected output : 6

Math.round()

Math.round() 함수는 입력값을 반올림한 수와 가장 가까운 정수 값을 반환한다.(반올림)

문법

Math.round(<x>)

x : 숫자

예시

Math.round(5.95); // expected output : 6

Math.abs()

Math.abs() 함수는 주어진 숫자의 절대값을 반환한다.

문법

Math.abs(<x>)

x : 숫자

예시

Math.abs(-5.95); // expected output : 5

2. String 객체 메서드

자주 사용하는 String 객체의 메서드 정리

toUpperCase()

toUpperCase() 메서드는 문자열을 대문자로 변환해 반환한다.(원래의 문자열에 영향을 주지 않는다)

"abcd".toUpperCase(); // expected output : ABCD

toLowerCase()

toLowerCase() 메서드는 문자열을 소문자로 변환해 반환한다.(원래의 문자열에 영향을 주지 않는다)

"ABCD".toLowerCase(); // expected output : abcd

includes()

includes() 메서드는 특정 문자열이 다른 문자열에 포함되어 있는지를 판별하고 결과를 true 또는 false 로 반환한다.

문법

str.includes(<searchString>, <position>)

searchString : 문자열에서 찾을 다른 문자열

position : searchString을 찾기 시작할 위치(default : 0)

예시

let sentence = "The quick brown fox jumps over the lazy dog";
sentence.includes("quick"); // expected output : true
sentence.includes("quick", 5); // expected output : false

indexOf()

indexOf() 메서드는 호출한 String 객체에서 주어진 값과 일치하는 첫 번째 인덱스를 반환한다.
일치하는 값이 없으면 -1을 반환한다.

문법

str.indexOf(<searchValue>, <fromIndex>)

searchValue : 찾으려는 문자열

fromIndex : 문자열에서 찾기 시작하는 위치를 나타내는 인덱스 값(default : 0)

예시

let sentence = "The quick brown fox jumps over the lazy dog";
sentence.indexOf("quick"); // expected output : 4
sentence.indexOf("quick", 5); // expected output : -1

concat()

concat() 메서드는 매개변수로 전달된 문자열을 문자열에 붙인 새로운 문자열을 반환한다.

문법

str.concat(<string>)

string : 합칠 문자열

예시

"Hello ".concat("I'm RyuCoding") // expected output : "Hello I'm RyuCoding"

slice()

slice() 메소드는 문자열의 일부를 추출하면서 새로운 문자열을 반환한다.

문법

str.slice(<beginIndex>, <endIndex>)

beginIndex : 추출 시작점인 0부터 시작하는 인덱스
만약 beginIndex가 str의 길이보다 크거나 같은 경우 slice() 메소드는 빈 문자열을 반환한다.

endIndex : 0부터 시작하는 추출 종료점 인덱스로 그 직전까지 추출한다.(인덱스 위치의 문자는 추출에 포함되지 않는다)
만약 endIndex가 생략되면 slice() 메소드는 문자열 마지막까지 추출한다.

예시

let sentence = "The quick brown fox jumps over the lazy dog";
sentence.slice(31); // expected output : "the lazy dog"
sentence.slice(31, 34); // expected output : "the"

split()

split() 메서드는 문자열을 지정한 구분자를 이용하여 여러 개의 문자열로 나누어 배열로 반환한다.

문법

str.split(<separator>, <limit>)

separator : 원본 문자열을 나누어야 할 부분을 나타내는 문자열
separator가 생략되거나 문자열에 등장하지 않을 경우 원본 문자열을 유일한 원소로 가지는 배열을 반환한다.
separator가 빈 문자열일 경우 문자열의 각각의 문자를 원소로 갖는 배열을 반환한다.

limit : 끊어진 문자열의 최대 개수를 나타내는 정수

예시

let sentence = "The quick brown fox jumps over the lazy dog";
sentence.split(" "); // expected output : ["The", "quick", "brown", "fox", "jumps", "over", "the", "lazy", "dog"]
sentence.split("", 4); // expected output : ["T", "h", "e", " "]

substring()

substring() 메소드는 문자열의 시작 인덱스로 부터 종료 인덱스 전 까지 문자열의 부분 문자열을 반환한다.

문법

str.substring(<indexStart>, <indexEnd>)

indexStart : 반환 문자열의 시작 인덱스

indexEnd : 반환 문자열의 마지막 인덱스로 해당 인덱스를 포함하지는 않는다.(마지막 인덱스 직전까지 반환)
indexEnd가 생략된 경우 문자열의 끝까지 모든 문자를 반환한다.

예시

let str = "Mozilla";
str.substring(2); // expected output : "zilla"
str.substring(2, 4); // expected output : "zi"

3. Array 객체 메서드

자주 사용하는 Array 객체의 메서드 정리

includes()

includes() 메서드는 배열이 특정 요소를 포함하고 있는지 판별하고 결과를 true 또는 false 로 반환한다.

문법

arr.includes(<valueToFind>, <fromIndex>)

valueToFind : 탐색할 요소

fromIndex : valueToFind를 찾기 시작할 위치(default : 0)

예시

let newArr = [1, 2, 3];
newArr.includes(2); // expected output : true
newArr.includes(2, 2); // expected output : false

indexOf()

indexOf() 메서드는 배열에서 지정된 요소를 찾을 수 있는 첫 번째 인덱스를 반환한다.
찾을 수 없으면 -1을 반환한다.

문법

arr.indexOf(<searchElement>, <fromIndex>)

searchValue : 배열에서 찾을 요소

fromIndex : 검색을 시작할 색인(default : 0)

예시

let beasts = ["ant", "bison", "camel", "duck", "bison"];
beasts.indexOf("bison"); // expected output : 1
beasts.indexOf("bison", 2); // expected output : 4
