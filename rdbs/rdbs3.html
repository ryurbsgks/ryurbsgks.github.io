MySQL 설치

1. MySQL 설치

apt-get을 이용해 mysql-server 설치

sudo apt-get update
sudo apt-get install mysql-server

2. MySQL 접속

다음의 명령어로 MySQL에 접속할 수 있다.

mysql -u root -p

MySQL을 처음 설치하면 root의 암호는 비어 있으므로 Enter키를 눌러서 MySQL에 접속한다.

비밀번호를 설정하고 나면 해당 비밀번호를 입력하여 MySQL에 접속한다.

비밀번호 변경

MySQL에서 root 사용자의 비밀번호를 변경하는 방법

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '<password>';

3. MySQL 연결

Node.js와 MySQL 연결 방법

const mysql = require("mysql");
const connection = mysql.createConnection({
  host: "<host>",
  user: "<user>",
  password: "<password>",
  database: "<db_name>"
});

connection.connect();

connection.query(<query_language>, function (error, results, fields){
  if(error){
    throw error;
  }
  console.log(results);
});

connection.end();

연결 옵션

host : 연결하려는 데이터베이스의 호스트 이름(default : localhost)
port : 연결할 포트 번호(default : 3306)
user : MySQL 사용자
password : MySQL 사용자의 비밀번호
database : 연결에 사용할 데이터베이스 이름
