async/await

1. async

function 앞에 async를 붙여 async function을 선언할 수 있다.

선언된 async function은 암시적으로 Promise를 사용하여 결과를 반환한다.

async function <functionName>() {
  <동작할 코드>
}

or

const <functionName> = async () => {
  <동작할 코드>
}

or

const <functionName> = async function () {
  <동작할 코드>
}

2. await

await 연산자는 Promise가 처리될 때까지 기다리는 연산자이다.

await 연산자는 async function 내부에서만 사용할 수 있다.

async function <functionName>() {
  let <variable> = await <Promise>
}

3. 에러 핸들링

await Promise가 거부되면 마치 throw문을 작성한 것처럼 에러를 반환한다.
이러한 await Promise의 에러는 try catch를 사용해 잡을 수 있다.

async function <functionName>() {
  try {
    <동작할 코드>
  } catch(err) {
    <에러 코드>
  }
}
