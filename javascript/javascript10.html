객체

1. 객체

객체는 key와 value로 구성된 Property들의 집합이다.

객체의 구조

중괄호({})를 이용해서 객체를 만들고 객체는 키(key)와 값(value)으로 구성된다.
키와 값사이는 콜론(:)으로 구분하며 키와 값 쌍은 쉼표(,)로 구분한다.

{<key>: <value>, <key>: <value>, <key>: <value>, ...}

2. 객체의 속성 조회

객체의 속성을 조회하는 방법에는 대표적으로 Dot notation과 Bracket notation이 있다.

Dot notation

조회하고자 하는 객체에 Dot(.)을 붙인후 키를 입력하면 해당 키의 값을 조회한다.

let user = {
  name: "Ryu",
  email: "email@email.com"
};
user.name // expected output : "Ryu"

Bracket notation

조회하고자 하는 객체에 대괄호([])을 붙인후 대괄호에 키를 문자열 형태로 입력하면 해당 키의 값을 조회한다.

let user = {
  name: "Ryu",
  email: "email@email.com"
};
user["name"] // expected output : "Ryu"

Bracket notation을 사용할때 키의 값이 동적으로 변하면(키가 변수일 때) 키를 문자열 형태가 아닌 변수 형태로 입력하면 해당 키의 값을 조회할 수 있다.

3. 객체 속성 추가 및 변경

Dot notation 또는 Bracket notation을 통해 객체에 새로운 key-value를 추가하거나 객체의 key에 접근하여 value를 변경할 수 있다.

let user = {
  name: "name",
  email: "email@email.com"
};
user.phone = "000-0000-0000";
user.name = "Ryu";
console.log(user); // expected output : {name: "Ryu", email: "email@email.com", phone: "000-0000-0000"}

4. 객체 속성 삭제

delete 연산자를 통해 객체의 key에 접근하여 해당 key-value를 삭제할 수 있다.

let user = {
  name: "name",
  email: "email@email.com"
};
delete user.email;
console.log(user); // expected output : {name: "name"}
