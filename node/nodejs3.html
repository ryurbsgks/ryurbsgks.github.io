Node.js 모듈

1. 모듈(module)

모듈이란 여러 기능들에 관한 코드가 모여있는 하나의 파일을 의미한다.

2. require()

Node.js 에서는 require() 함수를 사용하여 외부 모듈을 가져올 수 있다.

require() 함수의 매개변수로 파일 또는 디렉토리가 올 수 있다.
디렉토리를 입력하면 해당 디렉토리의 index.js를 가져온다.

const <moduleName> = require('<module>')

require() 함수를 사용한다고 모든 모듈을 가져올 수 있는건 아니다.
해당 모듈에서 exports 객체를 사용한 모듈, 함수, 변수 등만 require() 함수를 사용하여 가져올 수 있다.

3. exports

모듈은 독립적인 파일 스코프를 갖기 때문에 모듈 안에 선언한 모든 것들은 기본적으로 해당 모듈 내부에서만 참조 가능하다.
만약 모듈 안에 선언한 항목을 외부에 공개하여 다른 모듈들이 사용할 수 있게 하고 싶다면 exports 객체를 사용하여 exports 객체의 속성(property) 또는 메서드로 정의한다.

exports.<key_or_methodName> = <value_or_method>

4. module.exports

exports 객체가 원하는 속성(property) 또는 메서드를 필요에 따라 설정할 수 있다면 module.exports는 한곳에 정리하여 한번에 내보내는(export하는) 방법이다.

exports 객체는 exports 객체의 속성(property) 또는 메서드를 여러개 정의할 수 있지만 module.exports에는 하나의 값(원시 타입, 함수, 객체 등)만 할당할 수 있다.

객체를 할당하는 방식

module.exports = {
  <key>: <value>
}

함수를 할당하는 방식

module.exports = () => {
  <동작할 코드>
}

5. 3rd-party 모듈

3rd-party(서드 파티) 모듈은 해당 프로그래밍 언어에서 공식적으로 제공하는 빌트인 모듈(built-in module)이 아닌 모든 외부 모듈을 일컫는다.

3rd-party 모듈을 다운로드하기 위해서는 npm install 후 require() 함수를 사용하여 해당 모듈을 불러온다.

npm install <module> const <moduleName> = require('<module>')
