Sequelize Finders Methods

1. findAll

findAll 메서드는 정의된 모델의 모든 항목을 검색하는 표준 SELECT 쿼리를 생성한다.
여기에서 정의된 모델이란 sequelize.define() 되거나 init으로 Extending된 모델을 말한다.

const <model> = await <Model>.findAll();
console.log(JSON.stringify(<model>, null, 1));
// expected output : SELECT * FROM <Model>

2. findOne

findOne 메서드는 정의된 모델에서 조건에 맞는 첫번째 항목을 가져온다.

const <model> = await <Model>.findOne({
  where: {
    <query>
  }
});
if(<model> === null){ // (!<model>) 로도 표현 가능
  ...
}else{
  ...
}

query문은 객체의 형태로 표현한다.

3. findOrCreate

findOrCreate 메서드는 정의된 모델에서 조건에 맞는 항목을 찾으며 찾을 수 없는 경우 모델에 조건에 맞는 항목을 생성한다.

where 옵션은 항목을 찾기 위한 옵션이며 defaults 옵션은 조건에 맞는 항목이 없을 경우 생성되어야 하는 항목을 정의하는데 사용된다.
defaults 값이 없다면 where 옵션에 지정된 값을 사용한다.

const [<model>, <boolean>] = await <Model>.findOrCreate({
  where: {
    <query>
  },
  defaults: {
    <query>
  }
});
if(<boolean>){
  ...
}else{
  ...
}

<model>은 해당 모델의 인스턴스이다.

<boolean>은 조건에 맞는 항목을 찾거나 생성되었는지 여부를 나타내는 boolean 값이다.
