JSON

1. JSON

JSON(JavaScript Object Notation)은 데이터 교환을 위해 만들어진 객체 형태의 포맷이다.

JSON 포멧은 자바스크립트를 포함한 많은 언어에서 범용적으로 사용하는 유명한 포맷이다.

2. JSON 기본 규칙

JSON의 key와 value의 문자열은 반드시 큰따옴표("")를 사용해야 한다.

3. JSON 메서드

JSON의 메서드인 stringify() 메서드와 parse() 메서드에 대한 설명

stringify()

stringify() 메서드는 Object type을 JSON으로 변환하는 메서드이다.

JSON으로 변환된 객체의 타입은 문자열이다.

JSON.stringify(<objectType>)

parse()

parse() 메서드는 JSON을 Object type으로 변환하는 메서드이다.

JSON.parse(<JSON>)
